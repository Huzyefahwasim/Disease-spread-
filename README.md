# Disease Spread Simulator

## IT3016- SIMULATION AND MODELLING Semester Project

### Project Description
This project simulates the spread of an infectious disease using an SIR (Susceptible-Infectious-Recovered) model. It also includes Exploratory Data Analysis (EDA) on historical disease data and Machine Learning models to predict future infection rates. An interactive Streamlit dashboard is provided for visualization and simulation parameter tuning.

### Deliverables
- **Dataset Analysis**: Jupyter notebook for EDA (`notebooks/dataset_analysis.ipynb`)
- **Simulation**: Python script for SIR model simulation (`simulation/disease_sim.py`)
- **ML Models**: Training and prediction scripts (`models/`)
- **Interactive UI**: Streamlit dashboard (`app.py`)
- **Project Report**: Documentation and presentation outline (`docs/`)

### Setup Instructions
1. Clone the repository.
2. Create a virtual environment: `python -m venv venv`
3. Activate the virtual environment.
4. Install dependencies: `pip install -r requirements.txt`
5. Run the Streamlit app: `streamlit run app.py`
